## Grocery List Manager
## Description

This is a simple Java console application that allows a user to enter a grocery list of 6 items, ensuring no duplicates are entered. After all items are entered, the list is sorted alphabetically and displayed.

The program is case-insensitive when checking for duplicates (e.g., "Apple" and "apple" are treated as the same).

Features

Input up to 6 unique grocery items

Prevents duplicate entries

Displays the sorted grocery list

Case-insensitive duplicate detection
